# Voice-Based Concept Understanding Analyser

This project is a Streamlit-based application that evaluates spoken concept explanations using audio analysis, transcription, semantic similarity, and reporting.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud

1. Push this repository to GitHub.
2. Open Streamlit Cloud and create a new app.
3. Select this repository and set the main file as app.py.
4. Deploy.
